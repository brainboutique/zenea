Service Description

ZenEA is a self-hostable service for managing Enterprise Architecture assets. It allows seamless import of Application fact sheets from LeanIX and dynamic management in list and edit form. All data is purely stored in the file system to enable script-based im- and export as well as running scripted integrations or updates easily. Storage locations can be backed by GIT repositories and branches for version control, traceability and improved DevOps collaboration.
